Conheça um pouco sobre o Kep

Missão

Pretendemos levar o Kep Framework para todos e mostrar de forma clara para iniciantes, como é o desenvolvimento de uma framework e como podem aumentar a produtividade com Kep.

Equipe

Kep Framework foi iniciado por um desenvolvedor, com a mesma curiosidade de aprender como é um funcionamento de uma framework, utilizando como view o AngularJS.

Matuzalém S. Teles - GitHub | Facebook

Ínicio

Matuzalém S. Teles: O Kep Framework, começou quando estava em um desenvolvimento de um projeto próprio, onde estava trabalhando em uma grande atualização.

Precisava de uma framework para agilizar no desenvolvimento da aplicação. E como eu tinha duvidas sobre o funcionamento de uma Framework e como se trabalhar com a mesma, fiz um desafio a mim próprio, criar uma pequena framework que serviria para a criação de API, para ser consumida pelo AngularJS.

Comecei a estudar mais sobre framework e ler a documentação do Laravel, boa parte me inspirei na framework Laravel. De acordo a criação de framework e na atualização do meu projeto, fui colocando algumas ferramentas com a necessidade da aplicação.

Até então o momento não planejava disponibilizar a framework, então com o tempo fui gostando de programar mais ainda com a framework e decidir disponibilizar e continuar atualizanda a mesma, criando um site robusto, para a documentação simplificada e o propósito da mesma, para os iniciantes.
